Portafolio

Bienvenido. Este portafolio presenta una selección de mis proyectos en BI, análisis de datos, creación de dashboards y uso de herramientas.

Proyectos publicado en la Web

Consulta SECOP

Aplicación web para la consulta interactiva de datos de contratación pública en Colombia a través de la API del SECOP. Permite filtrar y visualizar información clave de manera eficiente.

Análisis de Contratistas Ganadores (SECOP)

Herramienta web que utiliza la API del SECOP para identificar los contratos adjudicados a un contratista específico, ya sea individualmente o en consorcio/unión temporal.

Dashboard de Avance - Transmilenio AV 68

Panel de control que presenta un resumen visual y completo del progreso del proyecto Transmilenio AV 68, basado en información pública.
